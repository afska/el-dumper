# ElDumper

## install

```bash
npm install
npm rebuild --runtime=electron --target=4.2.0 --disturl=https://atom.io/download/atom-shell --build-from-source
```

## run

```bash
npm run dev
npm run electron
```

## prettify

```bash
npm run prettier
```

## update dependencies

```bash
npm-check
```

## build

```bash
./build.sh
```

## editor

**Sublime Text** packages: (requires `PackageControl`)

```
- Babel
- EditorConfig
- JsPrettier
```
